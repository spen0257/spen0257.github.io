5 HTML tags

The Unordered List <ul> Tag
~ This is used when the listed points do not have a specific order of importance.
The Image Element <img> Tag
~ The image tag allows you to insert and display an image on the webpage.It allows a short description of the image as added information.
The Keyword <b> Tag
~ Allows you to define keywords, product names in a review, and lead sentences in a paragraph.
The Highlighting text <mark> Tag
~ This highlights the text referenced. For eg keywords on a Google results page.
The Citation <cite> Tag
~ Allows citation of quotations from another source.
