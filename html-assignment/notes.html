5 HTML tags

The Unordered List <ul> Tag
An unordered list is used when the items inside the list do not have a specific order or the order is not important. The unordered list starts with an opening tag and ends with a closing tag Inside the opening and closing tags we add each list item in its own opening list item and closing list item tags
The Image Element <img> Tag
The HTML element to display an image on the webpage is which is a singular tag. The tag needs two pieces of information one which image to link to and two a short description of the image.
The Keyword <b> Tag
defines a keyword, like product name in a review, a lead sentence in a paragraph.
The Highlighting text <mark> Tag
used to highlight a piece of text for reference. The keywords in a search results page, the current navigation item, etc.
The Citation <cite> Tag
a citation for another source, often used with quotations. A person name, a URL, a book, a movie title, etc.
